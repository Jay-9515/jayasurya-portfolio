Project Overview

This project demonstrates a complete reverse engineering workflow of a connecting rod cap component. The process involves scanning the connecting rod using a Scantech 3D scanner, converting the scanned data to a mesh format, redefining the origin, and then reconstructing the part back to its original CAD model form. After obtaining accurate dimensions from the scan, the part was reverse-engineered to create a final, production-ready CAD model of the connecting rod cap.

Project Location: University of Luxembourg

Process Workflow

1
3D Scanning

Used Scantech 3D scanner to capture high-resolution point cloud data of the physical connecting rod cap. This created a digital representation with sub-millimeter accuracy.

2
Mesh Processing

Converted point cloud to polygon mesh and refined the surface geometry. Redefined origin for proper alignment and optimized mesh topology for efficient CAD reconstruction.

3
CAD Reconstruction

Created parametric CAD model in Fusion 360 with precise dimensions retrieved from scan data. The final model is manufacturing-ready with fully editable features.

Technologies and Tools Used

Scantech 3D Scanner Fusion 360 3D Meshing Software CAD Modeling

Key Achievements

  • Successfully captured high-precision 3D scan data with sub-millimeter accuracy
  • Optimized mesh topology for efficient CAD reconstruction workflow
  • Created manufacturing-ready parametric model with fully editable features
  • Reduced traditional measurement time by implementing digital reverse engineering workflow
  • Successfully reconstructed original connecting rod cap geometry from scan data
